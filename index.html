PrismJS Code Highlighting
Hướng dẫn sử dụng thư viện mạnh mẽ để làm nổi bật code

Các bước thực hiện

  1. Copy một đoạn mã có sẵn dán vào trước thẻ head, phía cuối của thẻ
  2. Đọc thêm trong web prism, vì phần này sẽ được học sâu ở CSS
  3. Bạn có thể xem và dùng đoạn code trên Github của Gà lại lập trình để làm thử
